💻 ¿Cómo usar este material?

Nota

Este portal no ejecuta código directamente.

Para trabajar con los análisis de esta unidad:

  1. Descarga el repositorio o abre la carpeta en GitHub
  2. Guarda los archivos en tu computador
  3. Abre la carpeta unidad_i/ en RStudio
  4. Ejecuta el script principal (run_unidad1.R)

👉 Todos los análisis se realizan de forma local en R

📘 Selecciona un capítulo

Explora cada capítulo de la unidad con acceso directo a sus scripts, datos y resultados:

🔹 Capítulo 1 — Fundamentos de manipulación de datos

Versión reproducible y descargable del capítulo, diseñada para ejecutar los códigos completos que complementan el texto del libro y generar un reporte HTML de validación.

Contenido del capítulo:

  • R/ scripts completos (casos A y B + funciones auxiliares + guardado + render)
  • data/raw/ → plancton.xlsx, invert.xlsx
  • run_cap1.R → ejecución automática
  • limpiar_outputs.R → reinicio de salidas
  • reporte_cap1.qmd → validación en HTML

Ejecución:

source("run_cap1.R")

Esto generará:

  • tablas en outputs/tablas/

  • figuras en outputs/figuras/

  • reporte HTML en outputs/reportes/

  • 📁 Abrir carpeta del capítulo

🔽 Información de la unidad

Propósito

La Unidad I busca fortalecer competencias en el manejo, organización y exploración de datos ecológicos y ambientales, usando flujos de trabajo reproducibles en R. A lo largo de esta unidad, el usuario aprenderá a importar, depurar, transformar y visualizar bases de datos reales, como paso previo a análisis más avanzados de ecología de poblaciones y comunidades.
