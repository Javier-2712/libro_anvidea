# Capítulo 2. Visualización exploratoria de datos ecológicos

Material complementario para usuarios derivado del archivo `2_vis_explor.qmd` del libro **ANVIDEA**.

## Contenido

- `R/00_setup.R`: carga de paquetes y verificación básica.
- `R/01_casoA_plancton_exploracion.R`: flujo principal para visualización exploratoria del caso de mesozooplancton estuarino.
- `R/02_casoB_macroinvertebrados_exploracion.R`: flujo principal para ensamblajes fluviales.
- `R/03_funciones_auxiliares.R`: funciones simples para guardar figuras y renombrar taxones.
- `data/raw/LEAME_DATOS.txt`: guía para ubicar los archivos de entrada.
- `outputs/`: carpeta sugerida para guardar tablas y figuras.

## Archivos de entrada esperados

- `data/raw/plancton.xlsx`
- `data/raw/invert.xlsx`

## Orden sugerido de ejecución

1. `R/00_setup.R`
2. `R/01_casoA_plancton_exploracion.R`
3. `R/02_casoB_macroinvertebrados_exploracion.R`

## Alcance de esta versión

Esta es una versión **depurada para usuario**. Se retiró la mayor parte del contenido editorial del `.qmd` y se reorganizaron los procedimientos en scripts secuenciales más fáciles de ejecutar.
